# dtr-map

这是一个封装多个地图的公司内部插件


## 如何安装

- 安装项目依赖

```
npm install

```

## 如何使用

1. es6 `import` 导入

```javascript
import TdrMap from 'tdr-map'
```

2. commonjs `require` 导入

```javascript
const TdrMap =  require('tdr-map')
```

3. window 对象获取

```javascript
window.TdrMap
```


## npm script 命令说明

| 命令 | 功能 |
| ------------ | -------------- |
|  npm run dev |  启动开发环境 |
|  npm run build |  打包插件生产文件 |
